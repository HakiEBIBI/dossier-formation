## HTML
<Image src="/dossier-formation/HTML-LOGO.png" alt="HTML logo" width="100px" height="100px"/>


- **HTML** (HyperText Markup Language) est le langage de balisage utilisé pour structurer
  le contenu d'un site web. Il définit la structure de la page, comme les titres, les paragraphes,
  les images, les liens, et autres éléments. HTML est la base de toute page web.

## CSS
<Image src="/dossier-formation/CSS-LOGO.svg" alt="CSS logo" width="80px" height="80px"/>

- **CSS** (Cascading Style Sheets) est utilisé pour styliser et mettre en forme le contenu HTML.
  Il permet de définir l'apparence des éléments : couleurs, polices, espacements, alignements, etc.
  CSS permet de rendre les pages web attrayantes et de les adapter à différents écrans grâce au responsive design.
