## JavaScript

import { Image } from 'astro:assets';

<Image src="/dossier-formation/logo-javascript.svg" alt="Javascript logo" width="100px" height="100px"/>

JavaScript est un langage de programmation interprété utilisé principalement pour rendre les pages web interactives.
Il permet d'ajouter des fonctionnalités dynamiques comme la validation de formulaires, la modification du contenu sans
rechargement de page, et la création d'animations. Exécuté directement par le navigateur ou un moteur d'exécution comme
Node.js, il améliore l'expérience utilisateur en rendant les sites plus réactifs et interactifs.

## Code exemple
```javascript
function sayHello() {
  console.log("Hello, World!");
}
```

## TypeScript

<Image src="/dossier-formation/logo-typescript.svg" alt="TypeScript logo" width="100px" height="100px"/>

TypeScript est un langage de programmation qui est un sur-ensemble de JavaScript, ajoutant des fonctionnalités
de typage statique. Il permet de définir des types pour les variables, les fonctions et les objets, ce qui aide
à détecter des erreurs avant l'exécution. TypeScript est compilé en JavaScript, ce qui le rend compatible avec tous
les environnements JavaScript.


## Code exemple
```typescript
function sayHello(): void {
  console.log("Hello, World!");
}
```
